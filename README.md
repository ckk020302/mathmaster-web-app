# MathMaster – Web-Based Mathematics Learning Platform

MathMaster is a web-based system developed to support mathematics learning through interactive features, progress tracking, and collaborative study tools.

## Features
- Student and teacher role system
- Question bank management
- Classroom and study room features
- AI chatbot support for learning assistance
- Progress tracking and performance monitoring

## Technologies Used
- PHP (Laravel Framework)
- HTML, CSS, JavaScript
- MySQL Database
- Vite (Frontend build tool)

## Purpose
This project aims to enhance students’ learning experience by providing structured practice, collaboration, and intelligent assistance through an integrated web platform.
